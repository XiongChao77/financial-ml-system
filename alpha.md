# alpha 的来源是什么？

alpha 本质上来自：

市场不是完全有效的，或者说
不同信息处理者之间存在速度、认知、约束、结构差异。

常见来源可以分几类。

第一类：行为与认知偏差

人会：

反应慢
过度反应
从众
厌恶亏损
锚定

这些会产生：

动量
反转
事件后漂移
资金拥挤后的错价

这是很多中低频 alpha 的根。

第二类：结构性约束

很多参与者不是为了“纯收益最大化”交易，而是被约束着交易，比如：

指数基金被动调仓
风险平价被动去杠杆
保险/养老金有配置约束
做市商要库存管理
公募有风格、仓位、流动性限制

这些会制造可预测的流量和价格压力。

第三类：信息处理优势

谁能更早、更好地把信息转成价格判断，谁就有 alpha。

比如：

财报文本
新闻
另类数据
微观结构数据
基本面变量之间的非线性关系

这类地方最容易和机器学习结合。

第四类：风险补偿伪装成 alpha

有些“alpha”其实不是错价，而是你在收某种风险溢价，比如：

small cap
value
carry
illiquidity
crash risk

这类收益未必是“预测能力”，可能只是“承担别人不想承担的风险”。

所以实务里常要区分：

你赚的是 真 alpha，还是 beta / risk premium / hidden exposure。
